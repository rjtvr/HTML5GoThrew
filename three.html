HTML5 Web Storage

Html5 web storage, websites can store data on a users local computer
before Html5 javascript cookies are used to store data

Two Type of web storage

  • sessionStorage() : Destroyed when browser closed
  • localStorage() : Stored locally no expire date

  • Data is stored as key/valuepair

  • localStorage.setItem("key","value1");

  • alert(localStorage.getItem("key1"));

  • localStorage.removeItem("key1");

  • Removing all value : localStorage.clear()

  • For Session Storage : Instead of localStorage, sessionStroage is used

Example

  • localStorage.clear();

  • localStorage.setItem("b","a");

  • b: is key, a: is value
